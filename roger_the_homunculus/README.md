## To Start Node

* $ `git clone git@github.com:jrichocean/bprd.git`
* $ `cd bprd/roger_the_homunculus`
* $ `mix test` _(this will run whatever tests & compile app)_
* $ `iex --sname roger@localhost -S mix`
